每个局面存储为一个 $n\times m\times 2$ 的矩阵 $M$，$M_{i,j}$ 表示第 $i$ 行第 $j$ 列的状态：

1. $M_{i,j,0}$ 表示格子类型：
    - `0` 表示为山地或者空地。
    - `1` 表示塔。
    - `2` 表示别的玩家。

2. $M_{i,j,1}$ 表示格子颜色：
    - 若是山地或者空地则是 `-1`
    - 若是玩家则是 `[0,7]` 的颜色编号。

2. $M_{i,j,2}$ 表示格子的值：
    - 若是山地则是 `INF`。
    - 若是空地则占有空地的代价（空地是0，灰地是对应代价）。
    - 若是玩家则对应地或者塔的兵。

目前对齐，$n=32,m=32$。$M_{29,i,0/1}$ 一行表示颜色为 $i$ 的国家目前的兵力数和土地数。
